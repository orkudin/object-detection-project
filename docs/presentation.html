Разработка системы компьютерного зрения
для мониторинга ВПП с помощью БПЛА

UAV-CV-Navigator — модульный конвейер обнаружения, отслеживания объектов и адаптивного планирования маршрута беспилотного воздушного судна
🧠 YOLO11 + MobileNetV4 🎯 ByteTrack / BoT-SORT 🗺️ Adaptive Routing
🎯
Цели и задачи
Цель
Разработать отказоустойчивый конвейер компьютерного зрения для автономного мониторинга взлётно-посадочных полос с помощью БПЛА в режиме реального времени с адаптивной маршрутизацией.
Задачи
  1. 1 Реализовать детекцию объектов на базе YOLO11 с кастомным backbone MobileNetV4
  2. 2 Интегрировать мультитрекинг (ByteTrack / BoT-SORT) для устойчивого сопровождения целей
  3. 3 Спроектировать адаптивный навигатор с эвристической приоритизацией зон патрулирования
  4. 4 Обеспечить запись покадровой телеметрии (JSONL) для постполётного анализа
  5. 5 Создать Web UI (Gradio) для управления конвейером в реальном времени
  6. 6 Обеспечить модульность и Zero-Hardcode конфигурацию через YAML
